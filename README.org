#+AUTHOR: ayedero / osogyian

Itanna is an [[https://www.gnu.org/software/emacs/][Emacs]] distribution for electrical engineering, focused on four languages:

- **Python** — primary notebook language (org-babel / Jupyter)
- **Julia** — notebook language (org-babel)
- **C/C++** — firmware, embedded, simulation
- **Rust** — systems programming

* Quick Start

** One-liner install (recommended)

   #+begin_src sh
   curl -fsSL https://raw.githubusercontent.com/roqueando/itanna/main/itanna-quickinstall.sh | bash
   #+end_src

   Then restart Emacs. The welcome page will appear automatically.

** Manual install

   #+begin_src sh
   git clone https://github.com/roqueando/itanna.git ~/projects/itanna
   cd ~/projects/itanna
   ./itanna-quickinstall.sh
   #+end_src

   Or if you already have the repo:
   #+begin_src sh
   cd ~/projects/itanna
   ./install.sh
   #+end_src

3. Restart Emacs. The welcome page (~; h~) will appear automatically.

4. Create your first notebook: ~; o N~

* Virtual Environment

Itanna uses [[https://python-poetry.org/][Poetry]] to manage a dedicated Python virtual environment
at =~/.venv/= inside the project root. This venv contains all Python
dependencies (numpy, scipy, matplotlib, nuitka) and the =electrical=
package.

The venv is **automatically activated** when you open any file under
the =~/projects/itanna/= directory in Emacs. The activation sets:
- =VIRTUAL_ENV=, =PATH=, =PYTHONPATH=, =ITANNA_ROOT= environment vars
- =exec-path= so LSP and compile commands find the right Python
- Python sessions in org-babel use the venv Python

To manually re-activate: ~; V~

* The ~electrical~ Python Package

Located at =electrical/=, a pip-installable Python package providing EE
calculation modules and the executable builder.

** Converter Modules

- =electrical.converter.buck= — Buck (step-down) converter component selection

  Usage:
  #+begin_src python
  from electrical.converter.buck import design_buck
  results = design_buck(vin=12, vout=3.3, iout=2.0, fsw=300e3, vripple=0.01)
  #+end_src

** Executable Builder

- =electrical.executable.builder= — Build standalone binaries via Nuitka

  #+begin_src python
  from electrical.executable.builder import build_app, tkinter_template

  source = tkinter_template(title="My App", body="...")
  build_app(name="my-app", gui="tkinter", output_dir="~/Desktop/")
  #+end_src

See [[file:templates/hello-executable.org][hello-executable.org]] for a full walkthrough.

* Welcome Page

On startup, Itanna shows a welcome page (~; h~) with:
- Quick keybinding reference
- Environment status (Python version, installed packages, venv status)
- Links to templates and notebooks
- Quick reference cards for org-babel, EE tools, Rust, and CMake

The welcome page is an Org-mode buffer, so you can navigate links with
~TAB~ and execute code blocks with ~C-c C-c~.

* Dependencies

- **Emacs** 30+ (with tree-sitter support)
- **Python** 3.10+
- **Julia** 1.9+ (optional, for Julia notebooks)
- **Rust** (optional, for Rust development)
- **GCC/Clang** (for C/C++ compilation)

Optional Python packages:
- ~numpy~, ~scipy~ — scientific computing
- ~matplotlib~ — plotting
- ~nuitka~ — standalone executable builder
- ~PySide6~ — Qt GUI framework (alternative to Tkinter)
